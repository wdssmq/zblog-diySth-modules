## doc.md

用于 Z-BlogPHP diySth 插件的功能模块

https://github.com/wdssmq/zblog-diySth-modules

### 功能列表

| 文件夹名      | 功能简述                     | 其他             |
| ------------- | ---------------------------- | ---------------- |
| autoLoad      | 自动编译本插件规则           | 实际使用并不需要 |
| post301       | 文章增加跳转到指定链接功能   |                  |
| telForCmt     | 给评论增加手机号字段         | 需要修改模板文件 |
| ViewBookMarks | 用于查看浏览器导出的书签文件 |                  |
